Returns `NULL` when `kind` is `RMW_QOS_POLICY_INVALID` or an undefined kind.  The stringified version of the policy kind can be obtained doing the follwing conversion: RMW_QOS_POLICY_#lt;POLICY_KIND#gt; -#gt; lower_case(#lt;POLICY_KIND#gt;)  For example, the stringified version of `RMW_QOS_POLICY_DURABILITY` is "durability" and `RMW_QOS_POLICY_DEADLINE` is "deadline". <br/>#lt;hr#gt; Attribute          | Adherence ------------------ | ------------- Allocates Memory   | No Thread-Safe        | Yes Uses Atomics       | No Lock-Free          | Yes  \param[in] kind qos policy kind to be stringified. \return a null terminated string representing the policy kind, or \return `NULL` if kind is `RMW_QOS_POLICY_INVALID` or an undefined kind.<br/>
